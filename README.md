# project-management
 Project Management Application using Java and Spring
 
 
 ###Features:
 
 - Add Employee
 - Add Project
 - Set Many Employees to one Project
 - Set Many Projects to same Employee
 - Authentication 
 - CRUD Methods 
 - REST API using /api-app endpoints
 - Many more

###Technologies:
 - Java
 - Spring Boot, Spring MVC, Spring Security
 - PostgreSQL
 - HTML
 - CSS
 - JavaScript
